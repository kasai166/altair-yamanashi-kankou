# 山梨神社巡り観光サイト

山梨県内の神社を実際に訪問し、御朱印・季節の景観・混雑状況などを紹介する観光サイトです。
地域に密着した情報を発信しながら、山梨の神社を巡る楽しさを記録することを目的としています。

サイトURL
https://altair-yamanashi-kankou.com/

---

## 概要

本サイトでは山梨県内の神社を巡り、以下の情報を掲載しています。

- 御朱印情報
- 参拝時の混雑状況
- 季節ごとの景観
- 周辺観光地
- 実際の訪問レポート

例として、昇仙峡周辺にある金櫻神社の初詣レポートなどを掲載しています。
